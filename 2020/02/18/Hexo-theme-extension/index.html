Hexo-Tag-Cloud

Hexo 标签云插件：官方文档

安装流程

  1. 进入到 hexo 的根目录，然后在 package.json 中添加依赖: "hexo-tag-cloud": "2.1.*"
  2. 执行 npm install 命令
  3. 修改主题模板（详见下）
  4. 配置_config.yml（详见下）
